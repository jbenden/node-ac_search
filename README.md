node-ac_search
==============

A very fast AHO-Corasick String Search implementation.

Installation
------------

    $ npm install ac_search

Installation via source
-----------------------

    $ git clone https://github.com/jbenden/node-ac_search.git node-ac_search.git
    $ cd node-ac_search.git
    $ npm install
